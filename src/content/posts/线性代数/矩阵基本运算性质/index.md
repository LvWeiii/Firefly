### 1. 矩阵加法（最符合直觉）

设 $A, B, C$ 为同型矩阵：

- **交换律：** $A + B = B + A$
    
- **结合律：** $(A + B) + C = A + (B + C)$
    
- **转置性质：** $(A + B)^T = A^T + B^T$
    

---

### 2. 矩阵乘法（陷阱最多的地方 ⚠️）

设 $A, B, C$ 满足乘法条件：

- **结合律：** $(AB)C = A(BC)$（虽然顺序不能变，但你可以先算后面两个）
    
- **分配律：** $A(B + C) = AB + AC$
    
- **【极其重要】不满足交换律：** 一般情况下，$AB \neq BA$。
    
    - _笔试陷阱：_ 若 $AB=O$（零矩阵），不能推导出 $A=O$ 或 $B=O$。
        
- **【极其重要】不满足消去律：** 若 $AB = AC$ 且 $A \neq O$，**不能**推导出 $B = C$。
    

---

### 3. 转置运算（Transpose）

转置就像是矩阵的“镜像反射”：

- **双重转置：** $(A^T)^T = A$
    
- **【穿脱原则】：** $(AB)^T = B^T A^T$ （**顺序反转**，这是最常考的计算细节）
    
- **数乘转置：** $(kA)^T = k A^T$
    

---

### 4. 逆矩阵运算（Inverse）

设 $A, B$ 均为可逆方阵：

- **双重逆：** $(A^{-1})^{-1} = A$
    
- **【穿脱原则】：** $(AB)^{-1} = B^{-1} A^{-1}$ （**顺序反转**，和转置一样）
    
- **转置与逆可交换：** $(A^T)^{-1} = (A^{-1})^T$
    
- **数乘的逆：** $(kA)^{-1} = \frac{1}{k} A^{-1}$ ($k \neq 0$)
    

---

### 5. 行列式性质（Determinant）

这是将矩阵降维成一个“数”：

- **相乘的行列式：** $|AB| = |A| \cdot |B|$
    
- **转置不改变行列式：** $|A^T| = |A|$
    
- **【高频坑点】数乘的行列式：** $|kA| = k^n |A|$ （其中 $n$ 是矩阵的阶数。因为行列式每一行都提出来一个 $k$）
    
- **逆的行列式：** $|A^{-1}| = \frac{1}{|A|}$
    

---

### 6. 常用公式总结表（快速速记）

|**运算名称**|**符号**|**核心注意点**|
|---|---|---|
|**加法**|$A+B$|必须同型|
|**乘法**|$AB$|**顺序敏感**，不可随意交换|
|**转置**|$A^T$|$(AB)^T = B^T A^T$|
|**逆**|$A^{-1}$|$(AB)^{-1} = B^{-1} A^{-1}$|
|**伴随**|$A^*$|$AA^* = \\|A\$|

### 💡 华为笔试实战建议：

在做选择题时，如果看到选项里有 $(A+B)^2 = A^2 + 2AB + B^2$，**一定要警惕**。

因为在矩阵里：

$$(A+B)^2 = (A+B)(A+B) = A^2 + AB + BA + B^2$$

只有当 $AB = BA$（即 $A, B$ 可交换）时，那个公式才成立。而在一般情况下，它是错的。
