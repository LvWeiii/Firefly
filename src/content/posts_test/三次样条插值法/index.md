# 定义：

将插值区间[a,b]划分为n-1个子区间（n个节点），每个子区间内独立构造一个三次多项式（总共n-1个）。

函数必须严格通过所有给定节点。

# 光滑性要求

在该区间内一阶导数和二阶导数都要连续

# 目的

1、降低计算复杂度

2、避免高次多项式插值的振荡问题
如Runge现象：当使用高次多项式拟合等距节点数据时，区间端点附近的插值误差会随次数n增大而急剧增大，导致曲线剧烈振荡

# 特殊边界条件

自然样条：a与b处函数的二阶导数为0

clamped样条：给定两端的一阶导数值
