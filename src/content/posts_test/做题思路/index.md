## 一、 背包问题全家桶模板

所有背包问题的核心都在于：**利用 `dp[j]` 存储容量为 `j` 时的最优解。**

### 1. 0-1 背包（每个取 1 次）

- **特点**：每个物品选或不选。
    
- **核心**：**逆序**遍历容量。
    

Python

```
for item in items:
    for j in range(W, w - 1, -1): # 逆序：保证每个物品只用一次
        dp[j] = max(dp[j], dp[j - w] + v)
```

### 2. 完全背包（每个取无数次）

- **特点**：物品无限供应。
    
- **核心**：**正序**遍历容量。
    

Python

```
for item in items:
    for j in range(w, W + 1): # 正序：允许在同一轮中重复挑选
        dp[j] = max(dp[j], dp[j - w] + v)
```

### 3. 多重背包（每个取 $k$ 次）

- **特点**：每个物品有限定个数。
    
- **技巧**：**拆分法**。将 $k$ 个物品拆成 $k$ 个独立的 0-1 背包物品（若 $k$ 很大，用二进制拆分 $1, 2, 4 \dots$）。
    

Python

```
for item in items:
    for _ in range(k): # 暴力拆解：循环 k 次转为 0-1 背包
        for j in range(W, w - 1, -1):
            dp[j] = max(dp[j], dp[j - w] + v)
```

### 4. 分组背包（每组选 1 个）

- **特点**：多组物品，每组内互斥。
    
- **核心**：**三层循环**（组 $\to$ 容量 $\to$ 组内成员）。
    

Python

```
for group in groups:
    for j in range(W, -1, -1): # 逆序：保证这组里只能挑一个
        for w, v in group:
            if j >= w:
                dp[j] = max(dp[j], dp[j - w] + v)
```

## 二、 特殊限制与初始化

### 1. “恰好装满”型

- **最大价值**：`dp = [-float('inf')] * (W + 1)`, `dp[0] = 0`。
    
- **最小价值**：`dp = [float('inf')] * (W + 1)`, `dp[0] = 0`。
    
- **判定**：最后如果 `dp[W]` 仍为 `±inf`，说明无解。
    

### 2. “方案总数”型

- **初始化**：`dp = [0] * (W + 1)`, `dp[0] = 1`。
    
- **转移方程**：`dp[j] += dp[j - w]`。
    
- **顺序**：组合数先物品后容量；排列数先容量后物品。

### 2. 滚动数组优化

- **目的**：将 $O(N \times W)$ 空间降至 $O(W)$。
    
- **原则**：
    
    - 如果本行状态只依赖上一行：直接用一维数组，通过**遍历方向（正/逆）**控制依赖关系。
        
    - 如果逻辑复杂（如分组背包或必须选一个）：每一层循环开启时新建一个 `new_dp`，算完后 `dp = new_dp`。
        

### 3. 浮点数处理

- **技巧**：机考中遇到精度问题，**统一放大倍数转整数**。
    
    - `T = 0.3` $\to$ `int(0.3 * 100 + 0.5) = 30`。
        
    - 使用 `f"{ans:.2f}"` 格式化输出。

## 四、 华为机考做题套路

1. **读题扫关键字**：
    
    - “最多/最大” $\to$ 标准背包。
        
    - “恰好/必须” $\to$ 特殊初始化。
        
    - “每个只能选一次/多次/一组选一个” $\to$ 确定背包类型。
        
2. **草稿纸建表**：手动模拟前两行的状态转移，确认逻辑无误。
    
3. **处理输入**：华为是 ACM 模式，练习一下 `sys.stdin.read().split()`。
    
4. **空间溢出警告**：如果容量 $W > 10^5$，考虑是否需要滚动数组或更换状态定义（比如交换价值和重量的位置）。
    

祝你机考顺利！如果遇到拿不准的题目，记住：**先写暴力递归 + `lru_cache` 拿保底分，再考虑转一维 DP 冲满分。**加油！
